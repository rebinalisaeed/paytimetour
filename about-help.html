❓ یارمەتی (Help)

◀ گەڕانەوە بۆ سەرەکی

✅ تۆمارکردنی هاتن و دەرچوون

  • بۆ تۆمارکردنی هاتن، لە مێنووی "هاتن و دەرچوون" -> "تۆمارکردنی هاتن" هەڵبژێرە
  • کارمەند و شوێن و کاتەکە هەڵبژێرە، پاشان کلیکی "تۆمارکردنی هاتن" بکە
  • بۆ تۆمارکردنی دەرچوون، هەمان شێواز بەڵام "تۆمارکردنی دەرچوون" هەڵبژێرە
  • هەموو تۆمارەکان لە "لیستی هاتن و دەرچوون" دەبینرێت

⏰ ڕێکخستنی شەفت و گروپ

  • لە مێنووی "شەفت و گروپ" دەتوانیت گروپ و شەفتەکان پێناسە بکەیت
  • بۆ پێناسەکردنی شەفت، کاتی دەستپێک و کۆتایی و جۆری شەفت دیاری بکە
  • بۆ پێناسەکردنی گروپ، ناو و بەش و ئەندامەکان دیاری بکە
  • دەتوانیت خشتەی سوڕانەوەی شەفت بۆ گروپەکان پێناسە بکەیت

👥 بەڕێوەبردنی کارمەندان

  • لە مێنووی "زانیاری کارمەندان" دەتوانیت کارمەندانی نوێ زیاد بکەیت
  • کۆدی کارمەند بە شێوەی خۆکار زیاد دەبێت (دواین کۆد + ١)
  • هەموو زانیارییەکانی کارمەند (ناو، پلە، گروپ، مۆبایل) تۆمار بکە
  • دەتوانیت وێنەی کارمەند باربکەیت

💰 بەڕێوەبردنی مووچە

  • لە مێنووی "مووچە" دەتوانیت گرێبەست و زیادکردن/کەمکردنەکان دیاری بکەیت
  • فەرمانەکانی مووچە (خواردنەوە، گواستنەوە، باج) پێناسە بکە
  • مووچەی کارمەند بە شێوەی خۆکار حساب بکە
  • لیستی گرێبەستە بەسەرچووەکان ببینە و نوێیان بکەوە

💾 بەڕێوەبردنی داتابەیس

  • دەتوانیت لە داتاکان Backup و Restore بکەیت
  • داتاکان Export بکە بە CSV/JSON
  • داتا Import بکە لە فایلەوە
  • داتاکانی ناپێویست پاک بکەوە

📊 ڕاپۆرتەکان

  • ڕاپۆرتی دەوام، هاتن و دەرچوون، مۆڵەتەکان ببینە
  • دەتوانیت ڕاپۆرتەکان Export بکەیت بۆ CSV
  • فلتەر بەکاربهێنە بۆ دۆزینەوەی تۆمارە تایبەتییەکان
